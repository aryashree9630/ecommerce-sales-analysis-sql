# E-commerce Sales Analysis using SQL

## 📊 Project Overview
This project analyzes e-commerce sales data using SQL to extract meaningful business insights.

## 🔍 Key Insights
- Highest revenue generated from top-selling electronic products
- Sales peak observed in specific months
- Certain cities contribute significantly to total revenue
- Peak ordering hours identified

## 🛠️ Skills Used
- SQL (GROUP BY, ORDER BY, Aggregations)
- Data Analysis
- Data Visualization

## 📁 Dataset
E-commerce sales dataset from Kaggle
